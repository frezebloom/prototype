1. Устанавливаем node 
  https://nodejs.org/en/ 

2. Устанавливаем менеджер пакетов
  yarn (https://yarnpkg.com/lang/en/)

3. Скачиваем зависимости для api
  cd api && yarn install

4. Запускаем api
  node app.js

5. Скачиваем зависимости для фронта
  cd ../frontednt/ && yarn install

6. Запуск в dev режиме
  yarn dev

7. Сброка 
  yarn build
